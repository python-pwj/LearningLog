## 默认参数 必须指向不变对象！
```python
def power(x, n=2):
    s = 1
    while n > 0:
        n = n - 1
        s = s * x
    return s
>>> power(5)
25
>>> power(5, 2)
25
```

> 当函数有多个参数时，把变化大的参数放前面，变化小的参数放后面。变化小的参数就可以作为默认参数。  
> 使用默认参数有什么好处？最大的好处是能降低调用函数的难度。

## 可变参数 传入的参数个数是可变的，可以是1个、2个到任意个，还可以是0个。
```Python
def calc(*numbers):
    sum = 0
    for n in numbers:
        sum = sum + n * n
    return sum
>>> calc(1, 2)
5
>>> calc()
0


>>> nums = [1, 2, 3]
>>> calc(*nums)
14

```
