
## dict 使用键-值（key-value）存储，具有极快的查找速度。
```Python
>>> d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
>>> d['Michael']
95
```
- 判断key是否存在使用 in 或者 get()
```Python
>>> 'Thomas' in d
False

>>> d.get('Thomas')
>>> d.get('Thomas', -1)
-1 # 不存在返回None
```

> 和list比较，dict有以下几个特点：  
    1. 查找和插入的速度极快，不会随着key的增加而变慢；  
    2. 需要占用大量的内存，内存浪费多。
